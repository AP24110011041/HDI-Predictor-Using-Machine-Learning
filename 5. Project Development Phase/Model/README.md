# Trained Machine Learning Model

This folder contains the trained Linear Regression model used by the HDI Predictor web application.

## Model Information

- Model Name: HDI.pkl
- Algorithm: Linear Regression
- Framework: Scikit-learn
- Serialization: Pickle

The model is loaded by the Flask application to predict the Human Development Index (HDI) based on:
- Life Expectancy at Birth
- Mean Years of Schooling
- Gross National Income (GNI) per Capita
