# Human Development Index Prediction using Machine Learning

## Objective
Predict the Human Development Index (HDI) using Machine Learning.

---

## Technologies
- Python
- Flask
- Pandas
- NumPy
- Scikit-Learn
- HTML
- CSS

---

## Dataset
UNDP Human Development Report
Countries: 195

---

## Features
- Life Expectancy at Birth (2021)
- Mean Years of Schooling (2021)
- Gross National Income Per Capita (2021)

---

## Algorithm
Linear Regression

---

## Workflow
Dataset
↓
Data Cleaning
↓
Feature Selection
↓
Train-Test Split
↓
Model Training
↓
Model Evaluation
↓
Save Model
↓
Flask Application
↓
Prediction

---

## Results
The model predicts the HDI score and classifies countries into:
- Low
- Medium
- High
- Very High Human Development

---

## Conclusion
The project successfully predicts HDI using three important socio-economic indicators through a Linear Regression model.
