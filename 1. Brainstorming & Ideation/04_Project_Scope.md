# Project Scope

## In Scope

The project includes the following functionalities:

- Predict the Human Development Index (HDI) using Machine Learning.
- Perform data preprocessing and exploratory data analysis (EDA).
- Train and evaluate a Linear Regression model.
- Save the trained model for future predictions.
- Develop a Flask-based web application.
- Allow users to enter socio-economic indicators through a web interface.
- Display the predicted HDI score and category.

## Out of Scope

The following features are not included in this project:

- Live data collection from external APIs.
- Automatic model retraining.
- User authentication and account management.
- Cloud deployment.
- Mobile application support.
- Real-time HDI updates from international organizations.
