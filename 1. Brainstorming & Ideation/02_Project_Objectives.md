# Project Objectives

## Primary Objective

To develop a Machine Learning-based web application that predicts the Human Development Index (HDI) using socio-economic indicators with good accuracy.

## Secondary Objectives

- Collect and analyze the HDI dataset.
- Perform data preprocessing and cleaning.
- Train a Linear Regression model for HDI prediction.
- Evaluate the model using suitable performance metrics.
- Develop a Flask web application for user interaction.
- Display the predicted HDI score and its corresponding category.
- Create complete project documentation following the AI/ML project lifecycle.
