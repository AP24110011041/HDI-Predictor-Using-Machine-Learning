# Literature Survey

## Overview

Several studies have explored the prediction and analysis of the Human Development Index (HDI) using statistical methods and machine learning techniques. Researchers have used socio-economic indicators such as life expectancy, education level, and Gross National Income (GNI) to analyze human development.

## Existing Research

- Machine Learning algorithms improve prediction accuracy.
- Data preprocessing plays an important role in model performance.
- Linear Regression provides good results for continuous HDI prediction.
- Web-based applications improve accessibility for end users.

## Research Gap

Most existing studies focus on model development and analysis. Few provide a simple web application that allows users to predict HDI interactively using a trained machine learning model.

## Conclusion

This project combines Machine Learning and Flask to provide an easy-to-use web application for predicting the Human Development Index accurately and efficiently.
