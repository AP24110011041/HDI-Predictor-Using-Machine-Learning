# Software Requirements Specification (SRS)

## Project Title

Human Development Index (HDI) Prediction Using Machine Learning

---

## Purpose

The purpose of this project is to develop a Machine Learning-based web application capable of predicting the Human Development Index (HDI) using socio-economic indicators. The application provides users with a simple interface for entering input values and obtaining accurate HDI predictions.

---

## Scope

The application performs:

- Data preprocessing
- Machine Learning model training
- HDI prediction
- Web-based prediction using Flask
- Display of predicted HDI score and category

---

## Intended Users

- Students
- Researchers
- Policy Analysts
- Government Organizations
- Educational Institutions

---

## Software Requirements

- Python 3.x
- Jupyter Notebook
- Flask
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- HTML
- CSS
- JavaScript
- VS Code

---

## Hardware Requirements

- Windows 10/11
- Intel i3 or above
- Minimum 4 GB RAM
- 500 MB Free Storage

---

## Constraints

- Prediction depends on the quality of the dataset.
- Internet is required only for downloading dependencies.
- The system works on the trained model and does not automatically retrain itself.

---

## Assumptions

- Users provide valid input values.
- The trained model has acceptable prediction accuracy.
- Required Python libraries are installed.
