# System Requirements

## Hardware Requirements

- Processor: Intel Core i3 or higher
- RAM: Minimum 4 GB
- Storage: Minimum 500 MB free space
- Internet Connection (for package installation)

## Software Requirements

- Windows 10/11
- Python 3.x
- Visual Studio Code
- Jupyter Notebook
- Git
- Flask
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- HTML
- CSS
- JavaScript
