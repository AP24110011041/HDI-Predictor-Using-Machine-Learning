# User Stories

## Student

As a student, I want to predict the Human Development Index so that I can understand the relationship between socio-economic indicators and HDI.

## Researcher

As a researcher, I want to analyze HDI predictions so that I can study development trends.

## Policy Analyst

As a policy analyst, I want to estimate HDI quickly so that I can support decision-making.

## Government Organization

As a government official, I want an easy-to-use prediction system so that I can analyze development indicators efficiently.
