# Feasibility Study

## Technical Feasibility

The project uses widely available technologies such as Python, Flask, Pandas, NumPy, and Scikit-learn. Therefore, it is technically feasible.

## Economic Feasibility

The project uses open-source software and requires no additional licensing costs, making it economically feasible.

## Operational Feasibility

The web application is easy to operate and requires minimal user training.

## Schedule Feasibility

The project can be completed within the planned academic timeline.
