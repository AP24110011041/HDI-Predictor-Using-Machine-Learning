# System Architecture

## Overview

The Human Development Index (HDI) Prediction System follows a layered architecture that integrates Machine Learning with a Flask web application.

The system consists of the following layers:

1. User Layer
2. Frontend Layer
3. Flask Backend Layer
4. Machine Learning Layer
5. Model Storage Layer
6. Prediction Engine
